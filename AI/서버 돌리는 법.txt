### 올라마 로컬서버 돌리는법 법
-- pc에 ollama와 python 3.11이 설치되어 있어야 함!! --

1. run_all.bat 실행하기

2. 파이썬 flask 서버
    - 서버 url : http://127.0.0.1:5000
    - 엔드포인트 : "/action"

서버 실행 방법:

1. AI 디렉토리로 이동:
cd AI

2. 서버 실행:
python -m server.server

또는

python server/server.py

주의사항:
- 올라마 서버가 실행 중이어야 합니다
- 필요한 패키지가 설치되어 있어야 합니다 (fastapi, uvicorn)
- 포트 5000이 사용 가능해야 합니다
